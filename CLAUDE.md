## Repository Overview

This is a collection of standalone HTML/CSS/JavaScript web projects — no build system, no dependencies, no package manager. Each project lives in its own subfolder under `CCProjects/`.

## Current Projects

- **NewTest/index.html** — Two-player Tic Tac Toe game with score tracking
- **NewTest/sarcopenia-screen.html** — SARC-F sarcopenia risk screening tool branded for [29 Again Custom Fitness](https://29againfitness.com)

## Development

Since there is no build system, open any `.html` file directly in a browser to preview it. No compilation, bundling, or server required.

## Project Conventions

- Pure HTML/CSS/JS — no frameworks, no external libraries, no CDN imports
- All styles are written inline in `<style>` tags within the HTML file
- All scripts are written inline in `<script>` tags within the HTML file
- Each project is fully self-contained in a single `.html` file
